kkscope
=======

A porting of kscope 1.6.2 using Qt4 and KDE4.

Build
=====

1. make a build direcotry, such as "./build", and go into it.
   $ mkdir build
   $ cd ./build

2. use cmake to build project. "../husky/src" is the directory of CMakefile.
   $ cmake ../husky/src
   $ make

3. run the programe.
   $ ./husky
